# Project Overview 

This React web application is built using modern frontend technologies.
It includes routing, reusable components, a clean folder structure, and
responsive UI elements.

## Main Features
- Reusable components (Navbar, Footer ...)
- React Router navigation
- Custom hooks and services
- Organized folder structure for scalability
- Testing using Testing Library + Vitest/Jest

## Technologies Used
- React
- JavaScript (ES6+)
- React Router
- CSS 
- Vite / Create React App (depending on setup)
- Testing Library (for unit & component tests)
